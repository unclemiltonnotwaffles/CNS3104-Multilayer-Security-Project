## Overview
This project designs and simulates a multi-layer security architecture for small enterprises, integrating:
- VLAN segmentation with access control lists (East‑West security)
- Site‑to‑site IPsec VPN with AES‑256 (encrypted remote connectivity)
- Cisco IOS Intrusion Prevention System (North‑South security)

The entire network of ~60 devices is built in **Cisco Packet Tracer 9.0.0**.

## Repository Structure
- `PacketTracer/` – Final simulation file
- `Configs/` – Annotated CLI scripts for all routers and switches
- `Diagrams/` – Network topology in draw.io and PNG formats
- `Report/` – Full project proposal and final report
- `Presentation/` – Oral defense slides

## How to Use
1. Clone or download the repository.
2. Open the `.pkt` file with Cisco Packet Tracer 9.0.0.
3. Explore the configurations and test the security layers.
